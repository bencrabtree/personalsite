Recall Score Analysis¶

We have a relatively low recall score, though a relatively high precision score. I believe we can improve on both these numbers.

In [10]:
number_records_normal = len(df[df.Class == 0])

# Picking out indicies of fraud and normal transactions
fraud_indices = np.array(df[df.Class == 1].index)
normal_indices = np.array(df[df.Class == 0].index)

# Out of the normal indices we picked, randomly select half
random_normal_indices = np.random.choice(normal_indices, number_records_normal // 10, replace = False)
random_normal_indices = np.array(random_normal_indices)

# Appending the 2 indices
under_sample_indices = np.concatenate([fraud_indices, random_normal_indices])

# Under sample dataset
under_sample_data = df.iloc[under_sample_indices,:]

X_undersample = under_sample_data.iloc[:, under_sample_data.columns != 'Class']
y_undersample = under_sample_data.iloc[:, under_sample_data.columns == 'Class']
